- [x] Integrate Sanity CMS
- [x] Configure Daisy theme for Williamstown (light and dark)
- [ ] Basic layout for mobile with mobile menu
- [ ] Basic layout for desktop with menu at the top
- [ ] Homepage
- [ ] News
- [ ] Links to socials
- [ ] Sponsors
- [ ] Contact
- [ ] Club
